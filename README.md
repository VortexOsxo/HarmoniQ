## Base de donnees
La base de données est fournie via Google Drive.

- Téléchargez la base de données à partir du lien suivant : https://drive.google.com/file/d/1AChv-YwvDrE-nlYdT_aRSumKc571Cqxk/view?usp=sharing

- Une fois le téléchargement terminé, placez le fichier de base de données dans le dossier suivant du projet : `harmoniQ/harmoniq/db/`
